# TestTrain
TestTrain is an Integration Test Generator application allowing users to create vast amounts of test cases for APIs without the knowledge of test frameworks such as TestNG.

## Start Development Instance

`npm run develop`

## Build Application

### Windows
```npm run pack:win```

### Mac
```npm run pack:mac```

### Linux
```npm run pack:linux```
